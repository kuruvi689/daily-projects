# Habit Streak Tracker

**Date:** 2026-06-15
**Goal Area:** Knowledge System Building

## Purpose
Records daily habit completions and reports streaks, completion rate, and missed days.

## Usage

```bash
python main.py
python main.py --input sample_data.json
```

## What It Demonstrates
- Reading structured JSON input
- Producing a clear terminal summary
- Keeping project scope small enough for daily shipping

## Key Learning
Modeled simple personal data as JSON with useful summary metrics.
